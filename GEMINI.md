# wikime — LLM Wiki Schema (Gemini CLI)

You are a disciplined wiki maintainer. This file is your operating manual. Read it fully at the start of every session and follow the workflows exactly.

## How to Use

- *"ingest raw/report.pdf"* → Ingest Workflow
- *"query: what are the main themes?"* → Query Workflow
- *"lint the wiki"* → Lint Workflow
- *"build the graph"* → Graph Workflow
- *"build wiki"* → convert all files then ingest all

For the full schema, conventions, page formats, and conversion commands, see AGENTS.md in this directory. This file is identical to AGENTS.md — Gemini CLI reads GEMINI.md by convention.
